# 大乐透摇号模拟器

这是一个功能完整的大乐透摇号模拟器网页应用，支持单次摇号、批量摇号、手动输入号码、复制号码和统计号码出现次数等功能。

## 功能特性

- **单次随机摇号**：生成随机号码并模拟开奖过程
- **手动输入号码**：支持用户手动输入号码进行模拟
- **批量模拟**：支持批量生成号码并统计中奖情况
- **复制号码**：支持复制我的号码和开奖号码到剪贴板
- **号码统计**：统计所有摇号过程中号码出现的次数
- **真实的摇号体验**：开奖过程有动画效果，还原真实摇号体验

## 技术实现

- 使用纯HTML、CSS和JavaScript实现
- 响应式设计，适配不同屏幕尺寸
- 模块化代码结构，易于维护和扩展

## 如何使用

1. **单次随机摇号**：点击"单次随机摇号"按钮，系统会自动生成随机号码并模拟开奖过程
2. **手动输入号码**：点击"手动输入号码"按钮，在弹出的表单中输入前区和后区号码，然后点击"开始模拟"按钮
3. **批量模拟**：点击"批量模拟"按钮，输入模拟次数，然后点击"开始批量模拟"按钮
4. **复制号码**：点击"复制我的号码"或"复制开奖号码"按钮，将号码复制到剪贴板
5. **查看号码统计**：点击"查看号码统计"按钮，查看所有摇号过程中号码出现的次数

## 部署到Cloudflare Pages

### 步骤1：准备项目

确保项目包含以下文件：
- `index.html` - 主页面文件

### 步骤2：创建Cloudflare账户

如果还没有Cloudflare账户，请访问[Cloudflare官网](https://www.cloudflare.com/)注册一个账户。

### 步骤3：部署到Cloudflare Pages

1. 登录Cloudflare账户
2. 在仪表板中选择"Pages"
3. 点击"Create a project"
4. 选择"Connect to Git"或"Direct upload"
   - **Connect to Git**：如果项目已经在GitHub、GitLab或Bitbucket上，选择相应的仓库
   - **Direct upload**：直接上传项目文件夹
5. 配置部署设置：
   - 构建命令：`npm run build`（如果没有构建步骤，可以留空）
   - 构建输出目录：`/`（根目录）
6. 点击"Save and Deploy"
7. 等待部署完成，Cloudflare会提供一个默认的子域名

### 步骤4：访问部署后的应用

部署完成后，您可以通过Cloudflare提供的域名访问应用，例如：`https://your-project.cloudflarepages.com`

## 本地开发

1. 克隆项目到本地
2. 直接在浏览器中打开`index.html`文件
3. 开始使用模拟器
