# 大乐透摇号模拟器

这是一个功能完整的大乐透摇号模拟器网页应用，支持单次摇号、批量摇号、手动输入号码、复制号码、统计号码出现次数、历史记录、号码筛选和冷热分析等功能。

## 功能特性

### 核心功能
- **单次随机摇号**：生成随机号码并模拟开奖过程，带有真实动画效果
- **手动输入号码**：支持用户手动输入号码进行模拟
- **批量模拟**：支持批量生成号码（最多100次）并统计中奖情况
- **复制号码**：支持复制我的号码和开奖号码到剪贴板，格式为 `01 05 11 17 34 + 03 11`

### 高级功能
- **📜 历史记录**：自动保存摇号历史（最多100条），支持查看、清空和导出为TXT文件
- **🔍 号码筛选**：可排除不喜欢的号码，提高选号效率
- **🔥 冷热分析**：分析热门号码和冷门号码，帮助用户做出更明智的选择
- **📈 号码统计**：统计所有摇号过程中号码出现的次数，高频号码高亮显示

### 用户体验
- **真实的摇号体验**：开奖过程有动画效果，号码逐个弹出
- **现代化界面**：渐变配色、圆角卡片、流畅动画
- **响应式设计**：适配不同屏幕尺寸，支持移动端

## 如何使用

### 基本操作
1. **单次随机摇号**：点击"🎲 单次随机摇号"按钮，系统会自动生成随机号码并模拟开奖过程
2. **手动输入号码**：点击"✏️ 手动输入号码"按钮，在弹出的表单中输入前区（5个）和后区（2个）号码
3. **批量模拟**：点击"📊 批量模拟"按钮，输入模拟次数（1-100），查看批量结果

### 高级功能
4. **号码筛选**：点击"🔍 号码筛选"按钮，输入要排除的号码，应用后摇号将不会出现这些号码
5. **查看历史**：点击"📜 查看历史"按钮，查看所有摇号记录，支持导出为TXT文件
6. **冷热分析**：点击"🔥 冷热分析"按钮，查看热门和冷门号码分析
7. **号码统计**：点击"📈 号码统计"按钮，查看每个号码的出现频率

### 复制功能
- **单次摇号**：点击"复制我的号码"或"复制开奖号码"按钮
- **批量摇号**：点击"复制全部选号"或"复制全部开奖号"按钮，每组号码一行

## 号码格式说明

- **前区号码**：从01-35中选择5个号码
- **后区号码**：从01-12中选择2个号码
- **复制格式**：`前区号码 + 后区号码`，例如：`01 05 11 17 34 + 03 11`

## 部署到Cloudflare Pages

### 步骤1：准备项目

确保项目包含以下文件：
- `index.html` - 主页面文件

### 步骤2：创建Cloudflare账户

如果还没有Cloudflare账户，请访问[Cloudflare官网](https://www.cloudflare.com/)注册一个账户。

### 步骤3：部署到Cloudflare Pages

1. 登录Cloudflare账户
2. 在仪表板中选择"Pages"
3. 点击"Create a project"
4. 选择"Connect to Git"或"Direct upload"
   - **Connect to Git**：如果项目已经在GitHub、GitLab或Bitbucket上，选择相应的仓库
   - **Direct upload**：直接上传项目文件夹
5. 配置部署设置：
   - 构建命令：留空（纯静态页面）
   - 构建输出目录：`/`（根目录）
6. 点击"Save and Deploy"
7. 等待部署完成，Cloudflare会提供一个默认的子域名

### 步骤4：访问部署后的应用

部署完成后，您可以通过Cloudflare提供的域名访问应用，例如：`https://your-project.cloudflarepages.com`

## 技术实现

- **前端技术**：纯HTML5 + CSS3 + JavaScript（无框架依赖）
- **数据存储**：使用JavaScript变量存储历史记录和统计数据
- **动画效果**：CSS3动画 + JavaScript定时器
- **响应式设计**：CSS媒体查询 + Flexbox/Grid布局
